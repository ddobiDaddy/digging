## 🤖 AI(Ollama) 재활성화

룰 엔진으로 기본 흐름을 확정한 뒤 AI를 다시 켤 때의 체크리스트.

### 켜는 방법 (가장 빠른 경로)
1. https://ollama.com/download 에서 Ollama 설치 → 백그라운드 실행 (`ollama serve`)
2. 모델 다운로드: `ollama pull llama3.2` (또는 한국어가 더 안정적인 `qwen2.5:7b`)
3. `.env` 에서 `USE_AI=true` 로 변경
4. `npm run dev` 재시작 → 자동으로 AI 경로로 전환됨

코드는 이미 `server/api/analyze.post.ts` 의 `callOllama()` 함수에 보관되어 있어요. 룰 엔진과 동일한 JSON 모양을 반환하므로 프론트는 손댈 게 없습니다.

- [ ] 프롬프트 안정성: 작은 모델은 가끔 JSON을 깨뜨림 → JSON 스키마 검증 후 1회 재시도 로직
- [ ] 응답 스트리밍: 5~6개 체험을 받는 동안 사용자에게 진행 상태 보여주기
